# Store Inventory Manager
A command_line product management system built with python 

## Features 
- View full product catalogue 
- Add new products with input validation
- Rmove products by name
- Search product by ID
- View store statistics (cheapest , most expensive , average price)
- Data presists between sessions using JSON

## main file 
- python main.py

- Python dictionaries and lists
- Functions and modules 
- File handling and JSON
- Error handling and input validation
